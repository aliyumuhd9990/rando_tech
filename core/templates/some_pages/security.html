Security Policy

1. Overview

RandoTech is committed to protecting your personal data with industry-standard security measures.

2. Data Encryption and Storage

  • All sensitive data is encrypted both in transit (using SSL/TLS) and at rest.
  • We store data on secure servers with limited access.
  • Regular backups are made and stored securely.

3. Access Control


  • We enforce strict role-based access control (RBAC) to ensure only authorized personnel can access sensitive data.
  • User authentication includes secure password practices and optional two-factor authentication (2FA).

4. Security Monitoring

  • Our systems are continuously monitored for suspicious activities.
  • We maintain audit logs of key events to track access and changes.

5. Incident Response Plan

In the event of a data breach:

  • We promptly detect and assess the impact
  • Contain and resolve the issue
  • Notify affected users and authorities if necessary

6. User Responsibility

We encourage users to:

  • Use strong and unique passwords
  • Log out of accounts when finished
  • Report any suspicious activity immediately

7. Third-party Security

  • We only work with vendors who meet our security standards
  • Contracts require that third parties maintain confidentiality and data protection standards

8. Compliance and Certification

We aim to comply with relevant data protection regulations such as:

  • GDPR (Europe)
  • CCPA (California)
  • NDPR (Nigeria)

9. Training and Awareness

All employees undergo regular security awareness training, including phishing simulations and secure data handling practices.

10. Review and Updates

This policy is reviewed annually or whenever significant changes occur in technology or regulation.
